== Description ==

Verve SSL is a WordPress plugin that sets the WordPress login and Administration area to use HTTPS(SSL) and then after logging out of WordPress reverts back to HTTP automatically.  This plugin Also allows for HTTPS(SSL) while view your WordPress blog while logged in.  


== Installation ==



1. Extract ververssl.zip and Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
